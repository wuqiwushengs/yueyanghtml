# 悦养APP Web应用设计风格

## 设计理念

### 视觉语言
采用现代简约的设计风格，融合温暖的自然色调，营造专业而亲和的用户体验。整体风格类似于美团等成熟应用，注重功能性和易用性。

### 色彩方案
- **主色调**: 薄荷绿 (#10B981) - 代表健康与生机
- **辅助色**: 温暖橙色 (#F59E0B) - 传递温暖与活力  
- **背景色**: 浅灰绿 (#F0FDF0) - 营造舒适氛围
- **中性色**: 深灰 (#374151) - 确保良好的可读性
- **强调色**: 珊瑚粉 (#FB7185) - 用于重要提示和按钮

### 字体系统
- **标题字体**: Noto Serif SC - 传达专业与可信
- **正文字体**: Noto Sans SC - 现代简洁，易于阅读
- **装饰字体**: ZCOOL KuaiLe - 用于Logo和特殊标题

### 布局原则
- **卡片式设计**: 信息模块化，清晰分层
- **网格系统**: 12列响应式布局
- **留白运用**: 适当的间距提升阅读体验
- **一致性**: 统一的组件样式和交互规范

## 视觉效果

### 动画效果
- **页面切换**: 平滑的淡入淡出过渡
- **数据加载**: 骨架屏和进度指示器
- **交互反馈**: 按钮点击、表单提交的微动画
- **状态变化**: 任务完成、等级提升的庆祝动画

### 图标系统
- **风格**: 线性图标为主，填充图标为辅
- **尺寸**: 16px、24px、32px三个标准尺寸
- **颜色**: 与品牌色调保持一致
- **动效**: 重要操作添加微动画

### 组件设计
- **按钮**: 圆角设计，明确的层级关系
- **卡片**: 轻微阴影，圆角边框
- **表单**: 简洁的输入框，清晰的标签
- **导航**: 底部固定导航，图标+文字

### 数据可视化
- **图表配色**: 使用品牌色系的不同明度
- **动画**: 数据加载时的渐入动画
- **交互**: 悬停显示详细数据
- **响应式**: 适配不同屏幕尺寸

### 游戏化元素
- **进度条**: 渐变色设计，动态更新
- **成就徽章**: 金色、银色、铜色等级区分
- **植物图标**: 可爱的emoji风格，成长变化
- **等级系统**: 星星、皇冠等视觉元素

### 响应式设计
- **桌面端**: 三栏布局，充分利用屏幕空间
- **平板端**: 两栏布局，保持核心功能
- **移动端**: 单栏布局，底部导航优化

### 微交互设计
- **悬停效果**: 卡片轻微上浮，阴影加深
- **点击反馈**: 按钮按下效果，颜色变化
- **加载状态**: 旋转动画，进度指示
- **成功提示**: 绿色勾选动画
- **错误提示**: 红色警告图标，温和提醒

## 用户体验设计

### 信息架构
- **首页**: 今日概览 + 快速操作
- **任务页**: 养生任务列表 + 执行界面  
- **情绪页**: 情绪记录 + AI对话
- **我的页**: 个人数据 + 设置

### 交互流程
- **任务开始**: 选择场景 → 查看任务 → 开始执行 → 完成确认
- **情绪记录**: 选择情绪 → 输入文字 → AI回复 → 保存记录
- **植物养成**: 查看状态 → 互动操作 → 获得奖励 → 成长反馈

### 反馈机制
- **即时反馈**: 操作后立即显示结果
- **进度反馈**: 长时间操作显示进度
- **状态反馈**: 清晰的状态指示器
- **结果反馈**: 成功/失败明确提示
